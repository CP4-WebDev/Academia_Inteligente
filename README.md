# Academia Inteligente

> Site institucional de academia com calculadora de IMC, desenvolvido com React e Vite.

---

## Repositorio

🔗 [github.com/CP4-WebDev/Academia_Inteligente](https://github.com/CP4-WebDev/Academia_Inteligente)

---

## Tecnologias

- **React 19** — biblioteca de UI
- **React Router DOM** — navegacao entre paginas
- **Vite** — bundler e servidor de desenvolvimento

---

## Paginas

- **Home** — apresentacao da academia
- **Planos** — planos disponiveis
- **Sobre** — informacoes sobre a academia
- **Contato** — formulario de contato
- **Calculadora de IMC** — calculo de indice de massa corporal

---

## Como rodar localmente

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```
